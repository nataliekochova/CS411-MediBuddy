# MediBuddy
## Highlights
MediBuddy is a medication management app designed for people who may struggle to remember when to take their medications
Some of the features include:
- Quick searching for medications
- Ability to set dosage and medication type
- Scheduling reminders and notifications
- Tracking consistency
- Alerting contacts if medications haven't been taken

## Usage Instructions
### Requirements
- Java JDK 17+ (I'm using Java 25 locally, but 17+ should work, though you may have to change the pom.xml file locally)
- Maven (https://maven.apache.org/install.html)
### Setup
- git clone github.com/nataliekochova/CS411-MediBuddy/
- cd medibuddy
- check that java and mvn versions match with "java -version" and "mvn -version"
### Launch
- mvn clean javafx:run
